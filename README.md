# Team Sprint Schaatsen 🏒⛸️

Een interactief schaatsspel gemaakt met p5.js in de kleuren van SC Heerenveen!

## Over het spel

Team Sprint Schaatsen is een spannend racespel waarbij twee teams tegen elkaar strijden in een klassieke team sprint formaat. Elk team bestaat uit 3 schaatsers die na elkaar rijden:

- **Schaatser 1**: Rijdt 1 ronde
- **Schaatser 2**: Rijdt 2 rondes  
- **Schaatser 3**: Rijdt 3 rondes

De totale tijd van het laatste teamlid bepaalt de winnaar!

## Spelen

🎮 **Besturing:**
- Druk op **ENTER** om het spel te starten
- Hou de **SPATIEBALK** ingedrukt om te versnellen
- Laat de spatiebalk los om langzamer te gaan

🏁 **Doel:**
Versla Team B (computer) door zo snel mogelijk alle 6 rondes te voltooien met je drie schaatsers!

## Live Demo

## Technologie

- **p5.js** - Creative coding library voor grafische animaties
- **HTML5** - Canvas rendering
- **SC Heerenveen kleuren** - Authentieke clubkleuren (#0066cc)

## Lokaal uitvoeren

1. Clone deze repository
2. Open `index.html` in een moderne webbrowser
3. Start met schaatsen!

Geen build process of dependencies nodig - gewoon openen en spelen!
